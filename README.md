# formal-languages

Java project containing several applications about formal languages and compilation techniques.
- [Lexical Analyzer](https://github.com/Iri25/formal-languages/tree/main/LexicalAnalyzer) - Writing a lexical analyzer for a mini programming language (Java)
- [Finite Automate 1](https://github.com/Iri25/formal-languages/tree/main/FiniteAutomata1), [Finite Automate 2](https://github.com/Iri25/formal-languages/tree/main/FiniteAutomata2) - Writing a program for various finite automata

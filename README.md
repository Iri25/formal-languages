# formal-languages

Java project containing several applications about formal languages and compilation techniques.
- [Lexical Analyzer](https://github.com/Iri25/formal-languages/tree/main/LexicalAnalyzer) - Writing a lexical analyzer for a mini programming language (Java).
- [Finite Automate 1](https://github.com/Iri25/formal-languages/tree/main/FiniteAutomata1), [Finite Automate 2](https://github.com/Iri25/formal-languages/tree/main/FiniteAutomata2) - Writing a program for various finite automata.
- [FLEX](https://github.com/Iri25/formal-languages/tree/main/FLEX) - Writing a lexical analyzer using FLEX.
- [YACC-BISON](https://github.com/Iri25/formal-languages/tree/main/YACC-BISON) - Writing a program with YACC/BISON to check if the file to be analyzed is syntactically correct or not.
- [Syntactic Analysis](https://github.com/Iri25/formal-languages/tree/main/SyntacticAnalysis) - Writing a program that implements an syntactic analysis method.
- [Mini Translator](https://github.com/Iri25/formal-languages/tree/main/MiniTranslator) - Writing a mini translator.
